About This Project

Sentiment Analysis on College Reviews is a machine learning-based web application designed to help students and parents make informed decisions when choosing a college. By analyzing hundreds of reviews, our system provides a summarized sentiment — Positive or Negative — for each college.

📌 Why This Project?

Manually reading large volumes of college reviews is time-consuming. Our tool uses Natural Language Processing (NLP) to automate sentiment classification, making the decision-making process faster and more data-driven.

🛠 Technologies Used

  • HTML, CSS, JavaScript for frontend
  • Python with Pandas and Scikit-learn for data processing
  • Django for backend integration
  • Natural Language Processing (NLP) [BERT model] for sentiment analysis
